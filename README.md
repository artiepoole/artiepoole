### ⚙️ Current Projects

I'm primarily interested in bare-metal programming and am  currently exploring [Rust](https://rust-lang.org/) and [Zig](https://ziglang.org/). See below for my OS and Bootloader projects.

<details>
<summary><a href="https://github.com/artiepoole/ArtOS">ArtOS</a></summary>

My hobby x86_32 operating system written in C-style C++. ArtOS is a from-scratch OS project capable of playing DOOM (1993) in userspace, loaded from CD-ROM using asynchronous DMA drivers. Before writing this, I knew basically nothing about operating systems (e.g. I had no idea what a kernel was, or what user-space meant).

</details>

<details>
<summary><a href="https://github.com/artiepoole/Artinitium">ArtInitium</a></summary>

A multi-architecture bootloader written in [Zig](https://ziglang.org/) (0.16.2). ArtInitium aims to provide a clean, portable boot stage across multiple hardware targets.

</details>

### 🛠️ Day Job

At [Canonical](https://canonical.com/), I work as a low-level engineer, which means I spend my days close to the metal; fixing things like kernel drivers, graphics stack bugs, packaging errors and bootflow issues. FPGAd is my primary work project.
